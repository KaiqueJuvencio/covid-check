
## Build & Run

### Build

Linux/Mac
```
make build
```
Windows
```
docker-compose build --no-cache
```

### Run 

Linux/Mac
```
make all
```
Windows
```
docker-compose up backend
```

The server will be available on _http://localhost:3000_
